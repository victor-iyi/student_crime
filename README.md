# student crime tracker

Student's website tracker.

## Features

- Store information of all students studying in the school
  - Easy capturing and retrieval of stored student's info

- Capture suspects by recording list of students who did not sign in during school hours
  - display suspects on culprits dashboard (sex, matric no, name, time)
  - Retrieval of suspects in polynomial time

- Students information to be fetched by the students affairs
